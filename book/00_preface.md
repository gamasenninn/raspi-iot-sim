# 仮想環境で学ぶ Raspberry Pi IoT開発 実践ガイド

## まえがき

本書は、Raspberry Piの実機を持っていなくても、パソコン上の仮想環境だけでIoT（Internet of Things）開発を学べる実践的な入門書です。

「Raspberry Piを使ってIoT開発をやってみたい」と思ったとき、多くの人が最初にぶつかる壁があります。

- Raspberry Pi本体を購入する必要がある
- LEDやセンサー、ブレッドボードなどの電子部品を揃える必要がある
- 配線を間違えると部品を壊してしまうかもしれない
- そもそも電子工作の経験がない

本書では、これらの壁をすべて取り払います。Docker上で動く仮想Raspberry Pi環境「PI-CI」と、Pythonライブラリ「gpiozero」のシミュレーション機能を組み合わせることで、パソコン1台だけで本格的なIoTアプリケーション開発を体験できます。

### 本書で学べること

- Dockerを使った仮想Raspberry Pi環境の構築方法
- GPIOプログラミングの基礎（LED、ボタン、サーボモーターなど）
- 温度・湿度・気圧などのセンサーデータの取得と活用
- MQTTプロトコルを使ったIoTデータ通信
- REST APIによるデバイスの遠隔制御
- 複数デバイスの連携制御とステートマシン設計
- Webダッシュボードによるリアルタイム監視
- 自動テストによる品質保証
- 仮想環境から実機への移行方法

### 対象読者

- Pythonの基本的な文法を理解している方
- IoTやRaspberry Piに興味があるが、まだ触ったことがない方
- 電子部品を買い揃える前に、まずソフトウェアで試してみたい方
- IoTシステムの設計パターンを学びたいソフトウェアエンジニア

### 本書の特徴

**1. すべてのコードが実際に動く**

本書に掲載されているすべてのプログラムは、著者が実際に仮想Raspberry Pi環境上で実行し、動作を確認したものです。出力結果もすべて実際の実行結果を掲載しています。

**2. 段階的に複雑さが増す構成**

LED1個の点滅から始めて、最終的には複数のセンサーとアクチュエータが協調動作するスマートホームシステムまで、段階的にステップアップしていきます。

**3. 実機への移行を前提とした設計**

仮想環境で開発したコードは、最小限の変更で実際のRaspberry Piに移植できます。「仮想環境は学習用」ではなく、「仮想環境はプロトタイピング環境」として活用する方法を解説します。

### 動作環境

- Windows 10/11（macOS、Linuxでも動作可能ですが、本書ではWindowsを前提に解説します）
- Docker Desktop
- Python 3.11以上
- PowerShell
